# 99 · 故障排查索引

| 现象 | 处理 |
|------|------|
| `[跳过]` 无 API | 配置 `.env` 中 OPENAI 或 DASHSCOPE Key |
| `No module named '_shared'` | 从 `learn/` 根执行 `uv run python agentscope/...` |
| Import AgentScope 失败 | `uv sync --group agentscope-core` |
| Bridge curl 无输出 | 检查 8090 端口、`/health` |
| Tool 不被调用 | 强化 system_prompt + docstring When to use |
| 多 Agent 上下文丢失 | B 是否 `observe(A 的 reply)` |
| 旧教程 ReActAgent | 本仓库用 **Agent**（2.x） |
