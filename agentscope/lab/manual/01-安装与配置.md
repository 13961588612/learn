# 1 · 安装与配置

```bash
cd learn
uv sync --group agentscope-core
cp agentscope/.env.example agentscope/.env
```

## 环境变量

| 变量 | 说明 |
|------|------|
| `OPENAI_API_KEY` | 优先 |
| `OPENAI_BASE_URL` | 公司 Gateway |
| `OPENAI_MODEL` | 默认 gpt-4o-mini |
| `DASHSCOPE_API_KEY` | 备选通义 |

## 验证

uv run python agentscope/stage-1/02_msg_blocks.py
uv run python agentscope/stage-1/lab/scripts/01_first_reply.py
