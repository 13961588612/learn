# AG-UI · A2UI · CopilotKit 协议栈

| 名称 | 层级 | 职责 |
|------|------|------|
| CopilotKit | SDK + Runtime | React 组件、BFF、A2UI 中间件 |
| AG-UI | 传输 | SSE 双向事件（v2 默认） |
| A2UI | UI 声明 | Surface/Component/DataModel JSON |

**易混**：AG-UI 运输 A2UI，二者不竞争。

详见 [`../../base/copilotkit-a2ui/notes/ag-ui-a2ui-stack-map.md`](../../base/copilotkit-a2ui/notes/ag-ui-a2ui-stack-map.md)
